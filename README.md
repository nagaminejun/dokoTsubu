## スッキリわかるサーブレット&JSP入門第5版

14章までのどこつぶ (DokoTsubu) - つぶやき投稿SNSの作成と学習

### 📝 概要
Java Servlet/JSP を使用した、シンプルなつぶやき投稿型SNSアプリケーション\
ユーザー認証（ログイン）機能、投稿機能およびデータベース（H2 Database）と連携した投稿の一覧表示・保存機能を実装。

🚀 主な機能
ユーザー認証: データベースを参照したログイン・ログアウト機能。

つぶやき投稿: テキスト入力によるリアルタイム投稿。

投稿一覧表示: 投稿された内容をデータベースから取得し、時系列で表示。

🛠 使用技術
Language: Java 17

Framework/Library: Jakarta Servlet 6.0, JSP 3.0, JSTL 2.0

Database: H2 Database (Embedded mode)
